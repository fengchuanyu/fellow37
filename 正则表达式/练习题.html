电话号码正则

区号必填为3-4位的数字，区号之后用“-”与电话号码连接 例：0451-12345678-1234

电话号码为7-8位的数字 例：0451-12345678-1234

分机号码为3-4位的数字，非必填，但若填写则以“-”与电话号码相连接 例：0451-12345678-1234

手机号码正则表达式

正则验证手机号，忽略前面的0，支持130-139，150-159。忽略前面0之后判断它是11位的。

例：013889294444 ( true )

例：13889294444 ( true )

用户名正则表达式

正则验证用户名，包含数字、字母、下划线11位。

日期正则表达式

yyyy-mm-dd格式
